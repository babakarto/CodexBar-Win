Install in seconds

Clone the repo, pip install -r requirements.txt, run python codexbar.py. Or double-click CodexBar.bat — it handles everything. Build a standalone .exe with the included PyInstaller script.

Claude Code native

Built specifically for Claude Code on Windows. Spawns an interactive PTY session, sends /usage, and parses the real output — the same method the macOS version uses. Falls back to OAuth token or browser cookies automatically.

Privacy first

Everything runs locally on your machine. No telemetry, no external servers, no data collection. Your credentials never leave your computer. Reads only what Claude Code already stores locally.

Usage tracking

Session and weekly usage bars with live percentages. Color-coded from orange (low) to amber (medium) to red (high). Reset countdowns so you know exactly when your limits refresh.

Cost monitoring

Scans your local Claude Code JSONL logs to calculate real API costs. Shows today's spend and 30-day totals with token counts. All computed locally from your conversation logs.

Smart fallback chain

Tries CLI via PTY first (most accurate), then OAuth token from ~/.claude/.credentials.json, then browser cookies with DPAPI decryption, then JSONL logs. Always gets your data.

Requirements

Python 3.10 or later. Windows 10 or 11. Claude Code CLI installed and logged in (claude in your PATH). That's it — no admin rights needed, no browser extensions, no API keys to configure.
